# Class List Repository

Welcome to the class list repository!

In this exercise you'll practice using **branches** and **pull requests**.

## Instructions
1. **Fork** this repository.
2. Create a new **branch** in your fork.
3. Open the file `class-list.md`
4. Add your name to the list of students.
4. **Submit a pull request** to merge your changes to my original repo.
